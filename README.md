# 🏴‍☠️ La Isla del Tesoro - Videojuego

## Descripción General
Juego Metroidvania en 2D basado en la novela clásica *La Isla del Tesoro* de Robert Louis Stevenson. El jugador explora distintos escenarios de forma no lineal, resolviendo puzzles y enfrentando enemigos para avanzar en la historia.

## 📖 Historia
El juego se divide en 3 actos:

**Acto 1 - Conociendo al Capitán Bilbo**
- Jim Hawkins conoce al Capitán Bilbo y consigue el mapa de la Isla del Tesoro.

**Acto 2 - El Viaje**
- Jim y sus aliados consiguen un barco y navegan hacia la Isla del Tesoro.

**Acto 3 - La Isla**
- Al llegar, conocen a Ben Gunn. John Silver traiciona al grupo, se pierde gran parte del tesoro, pero se consigue suficiente oro para salvar la posada y pagar las deudas.

---

## 🕹️ Jugabilidad
- **Vista:** Lateral (side-scroller)
- **Desplazamiento:** Lateral y vertical no lineal
- **Herramientas:** Pistola, Mosquete y Llaves
- **Movimiento:** Caminar, correr, saltar (sin vuelo)
- **Mejoras:** Daño al disparar y puntería — se activan automáticamente al progresar

---

## 👾 Mecánicas
- Gravedad constante para objetos y personajes
- Colisiones sólidas
- Física de proyectiles
- Fricción dependiendo del terreno
- Fuerzas: Empuje, flotabilidad, viento
- Las reglas físicas no cambian durante el juego

## 🚀 Instrucciones para Ejecutar el Proyecto
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/mirkof13/la-isla-del-tesoro.git
   ```
2. Abrir el proyecto con **Unity** (versión recomendada: 2022.x o superior)
3. Abrir la escena principal desde `Assets/Scenes/`
4. Presionar **Play** en el editor de Unity
